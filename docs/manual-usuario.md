# 📖 Manual do Usuário - Bloqueador de Sites

Este guia explica como utilizar o sistema Bloqueador de Sites para restringir ou liberar o acesso a sites no computador.

## 🏁 Iniciando
Ao abrir o programa, você verá a tela principal com um campo para digitar sites, uma lista de sites cadastrados e botões de controle.

## 🛠️ Funcionalidades

### 1. Adicionar um Site
* Digite o endereço do site no campo indicado.
  * Exemplo: `facebook.com`
* Clique no botão **Adicionar**.
* O site aparecerá na lista logo abaixo.

### 2. Remover um Site
* Selecione o site desejado na lista.
* Clique no botão **Remover**.
* O site será excluído da lista.

### 3. Bloquear Sites
* Certifique-se de executar o programa como **Administrador**.
* Clique no botão **Bloquear**.
* Todos os sites da lista serão bloqueados imediatamente.

### 4. Desbloquear Sites
* Execute o programa como **Administrador**.
* Clique no botão **Desbloquear**.
* Os sites cadastrados serão liberados novamente.

### 5. Mensagens do Sistema
- O sistema exibe mensagens de sucesso ou erro.
- Caso o programa não seja executado como administrador, uma mensagem de erro será exibida.

---
*Dica: Utilize apenas o domínio do site (sem https ou www) para melhores resultados.*
